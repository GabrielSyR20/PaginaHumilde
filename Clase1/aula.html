Bienvenidos a la clase de Programacion 3

En esta clase vamos a aprender sobre HTML, CSS y JS

HTML es el lenguaje de marcado que se utiliza para estructurar el contenido de una página web

CSS es el lenguaje de estilos que se utiliza para dar formato y diseño a una página web

JS es el lenguaje de programación que se utiliza para agregar interactividad a una página web
